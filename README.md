# 🚀I am a mage!

A small 2D mage prototype game.  
You defeat enemies, do some simple platforming, and kill the boss.

 **Play the Game**  
👉 [Play on Itch.io](https://nihat21.itch.io/i-am-a-mage)

---

### ✨ Features
- Firing projectiles  
- Smooth physics-based movement  
- Handling collisions  
- Enemy AI such as attacking when in range, following player, dodging for boss, and edge detection to prevent falls  
- Simple UI with play and game over menus
- Implementation of OOP for simplicity
    

---

Made with Unity as a prototype to understand 2D mechanics .
